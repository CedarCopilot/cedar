# Chat Components

Cedar provides three pre-built chat components that you can use out of the box for different UI patterns. These components combine chat input, message rendering, and state management into ready-to-use solutions.

## Available Components

### CedarCaptionChat

A caption-style chat interface that appears at the bottom center of the screen, perfect for overlay-style interactions.

```tsx
import { CedarCaptionChat } from 'cedar-os-components';

function App() {
	return (
		<div>
			{/* Your main content */}
			<CedarCaptionChat
				dimensions={{
					width: 600,
					maxWidth: 800,
				}}
			/>
		</div>
	);
}
```

### FloatingCedarChat

A floating chat window that can be positioned on the left or right side of the screen with expand/collapse functionality.

```tsx
import { FloatingCedarChat } from 'cedar-os-components';

function App() {
	return (
		<div>
			{/* Your main content */}
			<FloatingCedarChat
				side='right'
				title='Assistant'
				collapsedLabel='How can I help you today?'
				dimensions={{
					width: 400,
					height: 600,
					minWidth: 350,
					minHeight: 400,
				}}
				resizable={true}
			/>
		</div>
	);
}
```

### SidePanelCedarChat

A side panel chat that pushes your main content to make room for the chat interface.

```tsx
import { SidePanelCedarChat } from 'cedar-os-components';

function App() {
	return (
		<SidePanelCedarChat
			side='right'
			title='Chat Assistant'
			dimensions={{
				width: 600,
				minWidth: 300,
				maxWidth: 800,
			}}
			resizable={true}>
			{/* Your main page content goes here */}
			<div className='p-8'>
				<h1>Your App Content</h1>
				<p>This content will be pushed to the side when chat opens.</p>
			</div>
		</SidePanelCedarChat>
	);
}
```

## Common Props

All chat components share some common configuration options:

- `dimensions`: Control width, height, and constraints
- `resizable`: Whether users can resize the chat interface
- `title`: The header title for the chat
- `companyLogo`: Optional logo component to display in the header
